Este projeto consiste no desenvolvimento de um site inspirado em plataformas de avaliação de animes, como o MyAnimeList, com o objetivo de disponibilizar informações detalhadas sobre diversas obras.

A aplicação apresenta dados como número de episódios, avaliações, autores e estúdios, utilizando uma API responsável por consumir e estruturar informações provenientes do MyAnimeList.

⚠️ Aviso: Este projeto pode não estar mais funcional devido à falta de atualizações e possíveis mudanças na API utilizada.


























Este projeto também deu origem ao apelido "ANIMIGAS" (Anime + Igas), criado por Moisés Fernandes Uchiha.
